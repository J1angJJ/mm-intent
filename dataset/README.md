# Dataset Sync Notes

This repository tracks the lightweight dataset files needed for development, including
`imu.csv` and the preprocessing scripts under `AR_Data_Process3.0/`.

Large raw video folders are intentionally not tracked by Git:

- `dataset/fisheye/`
- `dataset/HoloLens/`

Copy those two folders to the server separately with `scp` or `rsync` after cloning the
repository.
